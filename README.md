写代码写的累了？来放松一下吧，来用命令行查查天气，看看新闻，看个冷笑话，浏览图片，寻找属于程序员的浪漫~   
nodejs低版本无法使用图片功能

## Install

```
npm install irest-cli -g
```


## List

* `irest news`  看看新闻
* `irest joke`  来个笑话
* `irest weather [城市名称(选传)]`  看看天气
* `irest image [图片名称(必传)] [宽度(选传)]`  搜索一张图片看看
